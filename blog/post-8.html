Website Reflection

4/4/16

We are starting Phase 1 next week, so this will be my last blog post of Phase 0! This week has been a lot of review; I've been reviewing Ruby and Javascript and familiarizing myself with concepts we will explore in Phase 1 (like regular expressions, test driven development, and recursion). I spent the majority of today making edits and fixes to this website. Here are a few of my takeaways from building this site from scratch:

  1. It's a lot easier to format html correctly the first time, then to go back and fix it! The html on some of my blog pages was a bit disorganized, so I spent a few hours cleaning it up. Next time, I will be more mindful of making it perfect from the beginning.
  2. I did the initial CSS positioning of elements on this site in more traditional ways (using positioning and floats). I took a class in Flexbox at Girl Develop it about a month ago, and now have gone ahead and used Flexbox on many of the elements on this site. This has been a very helpful tool for me.
  3. Organization is so important when it comes to creating multiple HTML and CSS pages. It was a bit difficult to go back to my CSS pages and find the selector I was looking for. One of the DBC guides suggested ordering CSS selectors in alphabetical order, and I may go ahead and do that for clarity.
  4. I'm really looking forward to building a blog in Rails, so I don't have to keep creating new html pages each time I want to post a blog post!

See you in Phase 1- It will be a crazy/intense ride!
