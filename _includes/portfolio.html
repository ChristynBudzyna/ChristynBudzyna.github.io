Localeyes
A web app that connects tourists to day trips curated by locals. This was a group project built with John Collela, Hanna Taylor, and Elan Kvitko. We built Localeyes in Ruby on Rails, using the Materialize CSS framework, the Google Maps and Google Places APIs.
Another Openin'
A web app for theatre companies searching for their next musical. This is my ongoing passion project. I built the back end in Rails and the front end in React. I used Bootstrap as a CSS framework, and the Spotify API for sound clips. I am currently building a custom API of musicals that can be licensed for performance.
COMING SOON - Temperature Visualization
A visualization of the earth's temperature over time using Javascript and JQuery
