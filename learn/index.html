One-on-One Coding Classes for Beginners

About

Hi there! My name is Christyn and I'm a web developer, currently working at The Public Theater. I love to code and to teach others how to do it too. When I was learning how to code, I noticed that there are not a lot of opportunities for learning web development one-on-one. There are good resources online, but if you get stuck, there's no human to ask questions to. This summer, I'll be offering one-on-one classes in web development to beginners. These classes will be casual and fun. We'll be able to move at your pace and tailor the session to your skill level.

All the W's

Who: These sessions are great for all kinds of beginners- people who want to learn for fun or profit, college students deciding on a major, professionals considering a career change, people who work in other fields and need to create webpages/e-mails for work, people who want to understand the thought process of web developers on their team, etc.


What: I've designed each session to give you as much information as I can, so you feel confident continuing on your own. I believe the best way to learn how to code is to put your hands on the keyboard and type, so I'll do the talking, and you'll do the typing.


Where: The Astor Place Starbucks, unless you have a coffee shop (with wifi please!) that you prefer in Manhattan.


When: Monday or Tuesday evenings at 6:30/7:00pm


Why: Because coding is awesome! That's why!

Classes

HTML - 1 hr ($50)

HTML (HyperText Markup Language) is the most basic building block of the web; It’s the language that defines and describes every element of your website. It’s a great place to start if you’re new to coding. In this hour and a half crash course, I’ll teach you everything you need to know to get a webpage up and running with HTML.


Topics include

  • How to create the basic structure of a webpage
  • The most common HTML elements and their attributes (headers, paragraphs, images, lists, tables, etc.)
  • New features added with the newest version of HTML, HTML5 (section, article, nav, footer, video, audio etc.)
  • How to use Chrome developer tools and Sublime Text to debug your HTML
  • Adding classes and ids to your HTML tags
  • How to link your HTML file to a CSS file for styling

What you’ll need:

  • Your Laptop
  • Google Chrome
  • Sublime Text

CSS- 1 hr ($50)

CSS(Cascading Style Sheets) is the language you use to style your webpages. Once you know the basics of CSS, the design possibilities are endless.


Topics include:

  • How to target HTML elements for styling by type, class name, and id name
  • How to style text with color, size and font
  • How to add margin, borders, and padding to elements
  • How to use Chrome developer tools to try out different styling
  • The basics of positioning items on a page
  • How to create a horizontal navigation tab
  • How to write media queries to style for mobile and tablet view
  • How to link your stylesheet to your HTML file

What you’ll need:

  • Basic HTML knowledge
  • Your Laptop
  • Google Chrome
  • Sublime Text

JavaScript/JQuery- 1 hr ($50)

JavaScript is a powerful language that can be used to add interactivity to your website. JQuery is a helpful JavaScript library that makes it easier to interact with the elements on your page. Many concepts we cover in JavaScript can be used in other programming languages as well.


Topics include:

  • How to define and work with variables
  • Defining and calling functions
  • How to target HTML elements to add interactivity
  • The concept of the Document Object Model (DOM)
  • Creating mouse events (click, mouseover, etc.)
  • Manipulating classes and CSS in JavaScript
  • How to use the Chrome debugger tools to debug your JavaScript

What you’ll need:

  • Basic HTML & CSS knowledge
  • Your Laptop
  • Google Chrome
  • Sublime Text

Ruby- 1.5 hrs ($50)

This course is for beginners who are interested in learning a back-end programming language (Here’s a good explanation of front-end vs. back-end) Ruby is a great programming language for beginners, because it reads more like English than other languages. We will delve into a lot of concepts that apply to programming languages across the board.


Topics include:

  • How to define variables
  • How to work with different types of variables (integers, strings, booleans, arrays)
  • Wrapping code in methods and calling methods
  • Using built-in Ruby functions
  • Logging to the console and using pry to debug
  • Conditional statements
  • Looping and iterating

What you’ll need:

  • Your Laptop
  • Sublime Text
  • Ruby
